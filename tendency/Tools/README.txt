- Abstract_Parents - Абстрактные классы для установления договоренности
при создании классов построения графиков, вывода статистических данных,
конкатенации объектов типа pandas.DataFrame и подсчетов значений
объектов DataFrame;
- Singleton_Pattern - Реализация паттерна "Одиночка" через базовый класс.
Гарантирует, что у класса есть только один экземпляр, и предоставляет
к нему глобальную точку доступа;
- MVVM - Реализация паттерна "Модель - Вид Модель - Вид". Используется в
частных модулях процессов для реализации простенького интерфейса командной строки;
- Converter_to_xlsx - to_xlsx.py - кастомный конвертер в формат *.xlsx, несколько файлов, 
to_xlsx_pandas.py - pandas конвертер в формат *.xlsx в единый файл;
- Structure.png - файл с изображением структуры дирректорий проекта.
