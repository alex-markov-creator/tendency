Data - база данных проекта (БД).

Добавление объекта pandas.DataFrame,
(Показателя процесса):

1 Присвоить имя переменной объекту в файле database.py;
2 Документирование показателя в файле database.py;
3 Добавление имени переменной в список lst_name или lst_adhaesio;
4 Инкапсуляция переменной в файле __init__.py, т.е. присвоение
переменной __all__;
5 Импортирование объекта путем добавления имени переменной в
файле __init__.py, т.е. from .database import <ИМЯ ПЕРЕМЕННОЙ>;
6 Импортирование и обработка объекта в соответствующем модуле.

Тестирование БД:
pytest -v из корневого каталога
pytest -v Data - из корневого каталога
pytest -v tests_data/test_database.py - из каталога Data
