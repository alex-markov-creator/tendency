---------------------------------------------------------------------------------------------------------------------
adhaesio.py - модуль обработки данных по показателям качества характеризующих адгезию;
control_production.py - Модуль обработки данных по следующим показателям качества закупаемой(входящей) продукции;
files - результаты сохранения в файл;
log.txt - проситейшее логирование ошибок;
temp_data_graphics - последние используемые данные для подсчетов адгезии;
temp_data_control_production - последние используемые данные для подсчетов по показателям качества входящей продукции.

Проблемы при эксплуатации:
--------------------------

- Модуль adhaesio.py и gui.py:
	* данные за квартал формируются после формирования всей информации в одном жизненном цикле запуска GUI-интерфейса;
	* данные за квартал формируются длительное время без тайминга;
	* нет значений для толщины 1.5;
	* лишние экземпляры в памяти после сохранения.
